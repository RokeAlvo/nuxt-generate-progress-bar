# Nuxt-module for progress bar on generate logs

![demo](demo.gif =320x)

## Install

```nmp install -D nuxt-generate-progress-bar```
```javascript
// nuxt.config.js
module.exports = {
	buildModules: [
		'nuxt-generate-progress-bar',
    ],
}
```
```bash
# package.json
  "scripts": {
    "generate": "CONSOLA_LEVEL=1 nuxt generate"
  },
```
